# Negative keywords — Prima-Auto

Do ręcznego wklejenia w Google Ads Editor → Tools → Shared library →
Negative keyword lists, albo jako campaign-level negatives.

Zalecana konfiguracja (jeśli tworzysz Shared lists ręcznie):
  1. Lista 'prima_smieci_ogolne' (Negative Broad)  → zastosuj do 3 kampanii
  2. Lista 'prima_smieci_moto'    (Negative Broad)  → zastosuj do 3 kampanii
  3. Lista 'prima_brand_collision' (Negative Phrase) → zastosuj TYLKO do [Brand] Prima-Auto

Albo: po prostu zaimportuj `negative_keywords.csv` — campaign-level bez shared lists.
