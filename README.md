# primaauto

Projekt **asiaauto.pl** (marka: Prima-Auto) — repozytorium kontekstu, dokumentacji i skryptów.

## Struktura

- `CLAUDE.md` — główne instrukcje dla Claude Code (kontekst projektu)
- `docs/` — architektura, decyzje, kolejka zadań, runbooki
- `scripts/` — deploy, lint, diff, smoke test
- `data/` — eksporty, CSV, JSON
- `workflows/` — eksporty workflow n8n
- `.claude/` — konfiguracja Claude Code

## Produkcja

- **Domena:** asiaauto.pl
- **Hosting:** Hostido (Elara, host476470)
- **Plugin:** `~/domains/asiaauto.pl/public_html/wp-content/plugins/asiaauto-sync/`
- **Theme:** `~/domains/asiaauto.pl/public_html/wp-content/themes/asiaauto/`

## Praca

```bash
cd ~/projekty/primaauto
claude
```

Pełny kontekst dla Claude Code → patrz `CLAUDE.md`.
